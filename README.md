## Getting Started

### Installation
Usnig `yarn`
```
yarn add -D @ramble-inc/prettier-config
```
or using `npm`
```
npm i -D @ramble-inc/prettier-config
```

### Configuration
Create `.prettierrc` in project root directory
```
"@ramble-inc/prettier-config"
```
